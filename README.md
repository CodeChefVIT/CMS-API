## Instructions to run

```
$ git clone https://github.com/CodeChefVIT/CMS-API
$ cd CMS-API
$ pip3 install -r requirements.txt
$ virtualenv env
$ cd env
$ source bin/activate
$ python3 manage.py runserver
```
